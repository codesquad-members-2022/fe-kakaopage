## 요일 연재 TOP

- `Date` 인스턴스의 `get.day()`메서드를 이용해 오늘 날짜가 제일 처음 화면에 렌더링 되도록 함.
- 카카오 홈페이지에서 크롤링으로 받아온 데이터를 각각의 웹툰 카드로 매핑함.
- 클릭한 엘리먼트의 `innerText`로 요일의 정보를 얻어 데이터에서 웹툰의 연재 요일과 같은 요소만 필터링하여 렌더링 함.
